имеются следующие сущности:

Клиент - тот кто покупает продукцию у завода
	содержит: id, ФИО, адрес, номер телефона

Продукт - данные о заказанном продукте
	содержит id, наименование продукта, тип продукта

Поставщик - тот кто поставляет сырьё для производства
	содержит id поставщика, имя, адрес, номер телефона

Заказ - данные о заказе продукции клиентом или молокозаводом
	содержит номер заказа (ID), id клиента, id продукта, количество единиц заказанных продуктов, дата заказа

Поставка сырья - содержит данные о поставке товаров для молокозавода от поставщиков
	idПоставки, idПоставщика, idПродукта, кол-во продуктов, дата поставки
